Types de Données

En Lua, chaque valeur a un type. Découvrez les types fondamentaux et comment les utiliser.

Les types en Lua

Lua a 8 types de base, mais nous allons nous concentrer sur les 4 plus courants :

string

Texte, chaînes de caractères

number

Nombres (entiers et décimaux)

boolean

true ou false

nil

Absence de valeur

String (Texte)

Les strings représentent du texte. On les entoure de guillemets.

strings.lua 
local nom = "Alex"
local message = 'Bonjour !'
local citation = "Il a dit: 'Salut'"

-- Strings multilignes
local texte = [[
Ceci est une string
sur plusieurs lignes.
Très pratique !
]]

print(texte)

Opérations sur les strings

string_ops.lua 
local prenom = "Jean"
local nom = "Dupont"

-- Concaténation
local nomComplet = prenom .. " " .. nom
print(nomComplet)  -- Jean Dupont

-- Longueur d'une string
print(#prenom)  -- 4

-- Fonctions utiles
print(string.upper(prenom))  -- JEAN
print(string.lower(prenom))  -- jean
print(string.len(prenom))   -- 4

Number (Nombre)

Lua n'a qu'un seul type de nombre qui gère les entiers ET les décimaux.

numbers.lua 
local entier = 42
local decimal = 3.14159
local negatif = -100
local scientifique = 1e6  -- 1000000

-- Opérations mathématiques
print(10 + 5)   -- Addition: 15
print(10 - 5)   -- Soustraction: 5
print(10 * 5)   -- Multiplication: 50
print(10 / 5)   -- Division: 2
print(10 % 3)   -- Modulo (reste): 1
print(2 ^ 3)    -- Puissance: 8

-- Fonctions mathématiques
print(math.floor(3.7))   -- Arrondi inf: 3
print(math.ceil(3.2))    -- Arrondi sup: 4
print(math.abs(-10))     -- Valeur absolue: 10
print(math.random(1, 100)) -- Nombre aléatoire

Boolean (Vrai/Faux)

Les booleans n'ont que deux valeurs possibles : true ou false.

booleans.lua 
local estVivant = true
local aPerdu = false
local estAdmin = true

-- Les comparaisons retournent des booleans
local age = 18
print(age >= 18)   -- true
print(age == 21)   -- false
print(age ~= 18)   -- false (différent de)

-- Utilisation dans les conditions
if estVivant then
    print("Le joueur est en vie !")
end

Truthiness en Lua

En Lua, seuls false et nil sont "faux". Tout le reste est "vrai", même 0 et "" !

Nil (Rien)

nil représente l'absence de valeur ou une valeur inexistante.

nil.lua 
local x  -- Pas de valeur assignée = nil
print(x)  -- nil

local score = 100
score = nil  -- "Effacer" la valeur
print(score)  -- nil

-- Vérifier si nil
if x == nil then
    print("x n'a pas de valeur")
end

-- Utilisation courante
local joueur = nil  -- Sera défini plus tard

Vérifier le type

Utilisez type() pour connaître le type d'une valeur :

typeof.lua 
print(type("Hello"))   -- string
print(type(42))        -- number
print(type(3.14))      -- number
print(type(true))      -- boolean
print(type(nil))       -- nil
print(type(print))     -- function

-- Vérification de type
local valeur = "test"
if type(valeur) == "string" then
    print("C'est une string !")
end

Conversion de types

Parfois il faut convertir entre les types :

conversion.lua 
-- String vers Number
local texte = "42"
local nombre = tonumber(texte)
print(nombre + 8)  -- 50

-- Number vers String
local num = 100
local str = tostring(num)
print("Score: " .. str)  -- Score: 100

-- Conversion implicite
print("Age: " .. 16)  -- Lua convertit auto

Résumé

  • string : texte entre guillemets
  • number : tous les nombres (entiers et décimaux)
  • boolean : true ou false
  • nil : absence de valeur
  • type() : retourne le type d'une valeur
  • tonumber() et tostring() : conversion
