ClickDetector : Objets Interactifs

Apprenez à rendre vos objets cliquables ! Créez des boutons, des portes qui s'ouvrent, des objets à ramasser et plus encore.

Roblox Studio Requis

Cette leçon utilise l'API de Roblox. Ce code ne fonctionnera que dans Roblox Studio ou notre éditeur intégré, et non sur un interpréteur Lua PC standard.

Ajouter un ClickDetector

Le ClickDetector est un objet spécial qui permet de détecter quand un joueur clique sur une Part avec sa souris.

  1. Créez une Part dans le Workspace
  2. Cliquez sur le + à côté de la Part
  3. Cherchez et ajoutez ClickDetector
  4. Ajoutez aussi un Script dans la Part

Script de Base

bouton_simple.lua 
local part = script.Parent
local clickDetector = part.ClickDetector

-- Fonction appelée au clic
local function onClick(player)
    print(player.Name .. " a cliqué !")
    
    -- Changer la couleur aléatoirement
    part.BrickColor = BrickColor.Random()
end

-- Connexion
clickDetector.MouseClick:Connect(onClick)

Qui clique ?

L'événement MouseClick donne toujours le Player qui a cliqué comme premier paramètre. Très utile pour donner de l'argent ou des objets !

Exemple : Une Porte

Voici comment créer une porte qui s'ouvre et se ferme au clic :

porte.lua 
local porte = script.Parent
local clickDetector = porte.ClickDetector
local ouverte = false

clickDetector.MouseClick:Connect(function()
    if ouverte then
        -- Fermer la porte
        porte.Transparency = 0
        porte.CanCollide = true
        ouverte = false
    else
        -- Ouvrir la porte (invisible et traversable)
        porte.Transparency = 0.7
        porte.CanCollide = false
        ouverte = true
    end
end)

Exemple : Gagner des Points

Combinons ClickDetector et Leaderstats !

piece_a_ramasser.lua 
local piece = script.Parent
local cd = piece.ClickDetector

cd.MouseClick:Connect(function(player)
    -- Vérifier les leaderstats
    local stats = player:FindFirstChild("leaderstats")
    if stats then
        local coins = stats:FindFirstChild("Coins")
        if coins then
            coins.Value = coins.Value + 10
            
            -- Détruire la pièce
            piece:Destroy()
        end
    end
end)

Résumé

  • Ajouter un objet ClickDetector dans une Part
  • Utiliser l'event MouseClick
  • Le paramètre player permet de savoir qui a cliqué
  • Autres events : MouseHoverEnter, MouseHoverLeave
