Les Opérateurs

Les opérateurs sont les symboles qui permettent de manipuler les données. Mathématiques, comparaison et logique !

Opérateurs Arithmétiques

Ce sont les maths de base que vous connaissez déjà.

maths.lua 
local a = 10
local b = 3

print(a + b)  -- Addition: 13
print(a - b)  -- Soustraction: 7
print(a * b)  -- Multiplication: 30
print(a / b)  -- Division: 3.3333...
print(a ^ b)  -- Puissance (10 puissance 3): 1000
print(a % b)  -- Modulo (reste de la division): 1

Le Modulo (%)

Le modulo est très utile ! Par exemple : nombre % 2 vaut 0 si le nombre est pair, et 1 s'il est impair.

Opérateurs de Comparaison

Ils servent à comparer deux valeurs et retournent toujours un boolean (true ou false).

Symbole Signification
== Égal à
~= Différent de (Notez le tilde ~)
< Strictement inférieur
> Strictement supérieur
<= Inférieur ou égal
>= Supérieur ou égal
comparaison.lua 
local score = 100

print(score == 100)  -- true
print(score ~= 50)   -- true
print(score > 200)   -- false

Opérateurs Logiques

Ils permettent de combiner plusieurs conditions.

and (ET)

Vrai si les DEUX sont vrais

or (OU)

Vrai si AU MOINS UN est vrai

not (NON)

Inverse le résultat (Vrai devient Faux)

logique.lua 
local aLargent = true
local estNiveau10 = false
local estAdmin = true

-- AND : Les deux doivent être vrais
print(aLargent and estNiveau10)  -- false

-- OR : Un seul suffit
print(aLargent or estNiveau10)   -- true

-- Combinaison
-- (A l'argent ET niveau 10) OU est Admin
if (aLargent and estNiveau10) or estAdmin then
    print("Accès autorisé !")
end
