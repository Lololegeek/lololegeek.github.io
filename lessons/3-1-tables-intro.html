Introduction aux Tables

Les tables sont LA structure de données fondamentale en Lua. Elles peuvent représenter des tableaux, des dictionnaires, des objets et bien plus !

Qu'est-ce qu'une Table ?

En Lua, les tables sont l'unique structure de données complexe. Contrairement à d'autres langages, Lua n'a pas de tableaux, de listes, de dictionnaires ou d'objets séparés - tout est une table !

Une table peut contenir n'importe quel type de valeur comme clé et n'importe quel type de valeur comme valeur (sauf nil).

Le saviez-vous ?

Les tables Lua sont implémentées en tant que tables de hachage, ce qui les rend très performantes pour les recherches par clé !

Créer un Tableau (Array)

Un tableau (array) est une table dont les clés sont des nombres entiers séquentiels commençant par 1.

arrays.lua 
-- Créer un tableau vide
local monTableau = {}

-- Créer un tableau avec des valeurs
local fruits = {"Pomme", "Banane", "Orange"}

-- ATTENTION : Les indices commencent à 1 en Lua !
print(fruits[1])  -- Affiche : Pomme
print(fruits[2])  -- Affiche : Banane
print(fruits[3])  -- Affiche : Orange

-- Obtenir la taille d'un tableau
print(#fruits)    -- Affiche : 3

Indices à partir de 1 !

Contrairement à la plupart des langages (JavaScript, Python, C...) qui commencent à 0, Lua commence ses tableaux à l'index 1. Ne l'oubliez pas !

Manipuler les Tableaux

Lua fournit des fonctions utiles pour manipuler les tableaux via la bibliothèque table.

manipulation.lua 
local jeux = {"Adopt Me", "Blox Fruits"}

-- Ajouter à la fin
table.insert(jeux, "Murder Mystery 2")
print(#jeux)  -- 3

-- Ajouter à une position spécifique
table.insert(jeux, 2, "Jailbreak")
-- Résultat : {"Adopt Me", "Jailbreak", "Blox Fruits", "Murder Mystery 2"}

-- Supprimer le dernier élément
local dernier = table.remove(jeux)
print(dernier)  -- "Murder Mystery 2"

-- Supprimer à une position spécifique
local second = table.remove(jeux, 2)
print(second)  -- "Jailbreak"

Types Mixtes

Les tables Lua peuvent contenir différents types de valeurs dans le même tableau :

mixed.lua 
local mixte = {
    "texte",         -- string
    42,              -- number
    true,            -- boolean
    {1, 2, 3},       -- table imbriquée
    function()       -- fonction
        print("Hello!")
    end
}

print(mixte[1])     -- "texte"
print(mixte[2])     -- 42
print(mixte[4][2])  -- 2 (accès à la table imbriquée)
mixte[5]()          -- Appelle la fonction : "Hello!"

Résumé

  • Les tables sont la seule structure de données complexe en Lua
  • Les indices commencent à 1, pas à 0 !
  • {} crée une table vide
  • #table donne la taille d'un tableau
  • table.insert() ajoute un élément
  • table.remove() supprime un élément
  • Les tables peuvent contenir n'importe quel type de valeur
