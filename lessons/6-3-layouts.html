Layouts Flexibles

Apprenez à organiser vos éléments automatiquement sans calculer manuellement chaque position.

Pourquoi utiliser des Layouts ?

Calculer la position exacte de 10 boutons dans un menu est fastidieux. Si vous ajoutez un 11ème bouton, vous devez tout recalculer. Les UI Layouts font ce travail pour vous !

UIListLayout

C'est le plus utilisé. Il aligne tous les enfants d'une Frame en liste (verticale ou horizontale).

Propriétés utiles :

  • FillDirection : Vertical ou Horizontal.
  • Padding : L'espace entre chaque élément.
  • SortOrder : Ordre d'affichage (Name ou LayoutOrder).

UIGridLayout

Idéal pour les inventaires ou les boutiques. Il organise les éléments dans une grille avec des colonnes et des lignes.

Contrairement au ListLayout, le GridLayout impose sa propre taille (CellSize) à tous ses éléments enfants.

Les Constraints

Pour que votre UI reste parfaite sur tous les écrans :

  • UIAspectRatioConstraint : Garde les proportions (ex: un carré).
  • UIPadding : Ajoute des marges intérieures.
  • UISizeConstraint : Limite la taille min/max.
