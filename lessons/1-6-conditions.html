Les Conditions

Les conditions permettent à votre programme de prendre des décisions intelligentes en fonction des données.

La structure if

L'instruction if (si) exécute un bloc de code SEULEMENT si une condition est vraie. Elle se termine toujours par end.

if_simple.lua 
local estNuit = true

if estNuit then
    print("Allumer les lumières")
end

N'oubliez pas le "then" !

En Lua, il faut TOUJOURS mettre then après la condition. C'est une erreur très courante de l'oublier.

if ... else (Sinon)

else permet d'exécuter un code alternatif si la condition est fausse.

else.lua 
local score = 45

if score >= 50 then
    print("Tu as gagné !")
else
    print("Tu as perdu... Essaie encore !")
end

elseif (Sinon Si)

Pour tester plusieurs conditions à la suite, on utilise elseif.

elseif.lua 
local feu = "Orange"

if feu == "Vert" then
    print("Avancez")
elseif feu == "Orange" then
    print("Ralentissez")
elseif feu == "Rouge" then
    print("Stop !")
else
    print("Feu en panne ?")
end

Ordre des conditions

Lua vérifie les conditions dans l'ordre. Dès qu'une condition est vraie, il exécute son bloc et ignore le reste.

Conditions imbriquées

Vous pouvez mettre des if dans des if (mais attention à ne pas en abuser, c'est dur à lire !).

imbrique.lua 
local estConnecte = true
local estAdmin = false

if estConnecte then
    print("Bienvenue !")
    
    if estAdmin then
        print("Panneau Admin ouvert")
    else
        print("Menu Joueur ouvert")
    end
else
    print("Veuillez vous connecter")
end

Exercice Pratique

Contrôlez l'accès d'un joueur en fonction de son âge.

  1. Créez une variable age
  2. Si l'âge est supérieur ou égal à 13, affichez "Accès autorisé"
  3. Sinon, affichez "Accès restreint"

Résumé

  • if condition then : Début de la condition
  • elseif condition then : Alternative si la précédente est fausse
  • else : Si aucune condition n'est vraie
  • end : Fin obligatoire du bloc
