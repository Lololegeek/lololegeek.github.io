Paramètres & Retours Avancés

Allez plus loin avec les fonctions : valeurs par défaut, retours multiples et arguments variables.

Valeurs par défaut

En Lua, si vous appelez une fonction sans lui donner tous ses paramètres, les manquants vaudront nil. On peut utiliser ça pour créer des valeurs par défaut.

default.lua 
local function saluer(prenom, message)
    -- Si message est nil, on utilise "Bonjour"
    message = message or "Bonjour"
    
    print(message .. ", " .. prenom .. " !")
end

saluer("Alex")           -- Bonjour, Alex !
saluer("Sarah", "Salut") -- Salut, Sarah !

Retours multiples

Une fonction Lua peut retourner plusieurs valeurs à la fois. C'est une fonctionnalité très puissante !

multi_return.lua 
local function getPos()
    -- Retourne x, y, z
    return 10, 25, 5
end

local x, y, z = getPos()

print("X:" .. x .. " Y:" .. y)

Arguments variables (...)

Parfois on ne sait pas combien d'arguments on va recevoir. On utilise ... (les trois points).

varargs.lua 
local function toutAfficher(...)
    local args = {...}  -- Mettre tous les arguments dans une table
    
    for i, v in ipairs(args) do
        print("Argument " .. i .. ": " .. tostring(v))
    end
end

toutAfficher("Pomme", 42, true)

Résumé

  • Utilisez x = x or val pour les valeurs par défaut
  • Les fonctions peuvent retourner plusieurs valeurs : return a, b
  • ... permet d'accepter un nombre infini d'arguments
