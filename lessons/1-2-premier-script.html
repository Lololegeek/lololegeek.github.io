Premier Script

Écrivez votre premier programme Lua et apprenez à utiliser la fonction print() pour afficher du texte.

La fonction print()

La première chose que tout programmeur apprend est d'afficher du texte. En Lua, on utilise la fonction print() pour afficher des messages dans la console.

hello.lua 
print("Bonjour le monde !")

Ce simple programme affiche "Bonjour le monde !" dans la console. C'est le programme traditionnel que tous les programmeurs écrivent en premier !

Console Roblox

Dans Roblox Studio, la console s'appelle "Output". Vous pouvez l'ouvrir via View → Output.

Les Strings (Chaînes de caractères)

Le texte entre guillemets s'appelle une string (chaîne de caractères). En Lua, vous pouvez utiliser des guillemets doubles "..." ou simples '...'.

strings.lua 
-- Guillemets doubles
print("Ceci est une string")

-- Guillemets simples
print('Ceci aussi est une string')

-- Les deux sont équivalents !

Afficher des nombres

Vous pouvez aussi afficher des nombres sans guillemets :

nombres.lua 
print(42)
print(3.14159)
print(100 + 50)  -- Affiche 150

Les Commentaires

Les commentaires sont des notes dans votre code qui ne sont pas exécutées. Ils servent à expliquer ce que fait votre code.

commentaires.lua 
-- Ceci est un commentaire sur une ligne

print("Ce code s'exécute")  -- Commentaire en fin de ligne

--[[
    Ceci est un commentaire
    sur plusieurs lignes.
    Très utile pour les longues explications !
]]

print("Le programme continue")

Bonne pratique

Commentez toujours votre code ! Vous (et les autres) vous remercierez plus tard.

Concaténation

La concaténation permet de combiner des strings ensemble. En Lua, on utilise l'opérateur .. (deux points).

concatenation.lua 
print("Bonjour" .. " " .. "le monde !")
-- Affiche: Bonjour le monde !

local nom = "Alex"
print("Salut, " .. nom .. " !")
-- Affiche: Salut, Alex !

-- Concaténer avec des nombres
local age = 16
print("J'ai " .. age .. " ans")
-- Affiche: J'ai 16 ans

Exercice Pratique

Maintenant, c'est à vous ! Essayez de créer un programme qui :

  1. Affiche votre nom
  2. Affiche votre âge
  3. Affiche un message de bienvenue personnalisé
Ouvrir l'Éditeur
solution.lua (essayez d'abord !) 
-- Ma présentation
local monNom = "Alex"
local monAge = 16

print("Nom: " .. monNom)
print("Âge: " .. monAge)
print("Bienvenue, " .. monNom .. ", tu as " .. monAge .. " ans !")

Résumé

  • print() affiche du texte dans la console
  • Les strings sont entourées de guillemets "..." ou '...'
  • Les commentaires commencent par --
  • L'opérateur .. concatène (combine) des strings
