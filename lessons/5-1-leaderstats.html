Leaderstats : Système de Score

Créez un système de statistiques qui s'affiche dans le leaderboard Roblox. Parfait pour les points, l'argent, les niveaux...

Qu'est-ce que les Leaderstats ?

Les leaderstats sont un système intégré à Roblox qui affiche automatiquement les statistiques des joueurs dans le tableau de scores (appuyer sur TAB dans le jeu).

Le nom est important !

Le folder DOIT s'appeler exactement "leaderstats" (en minuscules) pour que Roblox le reconnaisse automatiquement.

Script Complet

Créez un Script dans ServerScriptService :

LeaderstatsScript.lua 
local Players = game:GetService("Players")

local function onPlayerAdded(player)
    -- Créer le folder leaderstats
    local leaderstats = Instance.new("Folder")
    leaderstats.Name = "leaderstats"
    leaderstats.Parent = player
    
    -- Créer une stat "Coins"
    local coins = Instance.new("IntValue")
    coins.Name = "Coins"
    coins.Value = 0
    coins.Parent = leaderstats
    
    -- Créer une stat "Level"
    local level = Instance.new("IntValue")
    level.Name = "Level"
    level.Value = 1
    level.Parent = leaderstats
    
    print(player.Name .. " : leaderstats créés!")
end

-- Connecter l'événement
Players.PlayerAdded:Connect(onPlayerAdded)

Modifier les Stats

modifier_stats.lua 
-- Accéder aux stats d'un joueur
local player = game.Players:FindFirstChild("NomDuJoueur")

if player then
    local leaderstats = player:FindFirstChild("leaderstats")
    
    if leaderstats then
        local coins = leaderstats.Coins
        
        -- Ajouter des coins
        coins.Value = coins.Value + 10
        
        -- Enlever des coins
        coins.Value = coins.Value - 5
        
        -- Définir une valeur exacte
        coins.Value = 100
    end
end

Types de Values

IntValue

Nombres entiers (0, 1, 100...)

NumberValue

Nombres décimaux (1.5, 3.14...)

StringValue

Texte ("VIP", "Noob"...)

BoolValue

Vrai/Faux (true/false)

Résumé

  • Créer un Folder nommé exactement "leaderstats"
  • Le parent doit être le Player
  • Ajouter des IntValue/StringValue dans le folder
  • Le nom de la Value = nom affiché dans le leaderboard
  • Modifier avec .Value
