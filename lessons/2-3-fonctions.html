Les Fonctions

Les fonctions permettent d'organiser votre code en blocs réutilisables. C'est la clé pour créer des programmes complexes mais organisés.

Qu'est-ce qu'une fonction ?

Imaginez une fonction comme une recette de cuisine. Vous lui donnez un nom (ex: "Faire des crêpes"), et chaque fois que vous voulez des crêpes, vous dites simplement "Faire des crêpes" au lieu de relire toute la recette.

fonction.lua 
-- 1. Définition de la fonction
local function saluer()
    print("Bonjour !")
    print("Comment allez-vous ?")
end

-- 2. Appel de la fonction
saluer()
saluer()  -- On peut l'appeler autant de fois qu'on veut !

Local vs Global

Comme pour les variables, utilisez local function pour que votre fonction ne soit accessible que dans le script actuel. C'est plus propre et plus rapide !

Pourquoi utiliser des fonctions ?

  • Réutilisation : Écrivez le code une fois, utilisez-le partout
  • Organisation : Découpez un gros problème en petits morceaux
  • Lisibilité : tuerMonstre() est plus clair que 50 lignes de code de combat

Paramètres (Arguments)

Les fonctions sont encore plus puissantes quand on leur donne des données. On appelle ça des paramètres ou arguments.

params.lua 
local function direBonjour(prenom)
    print("Salut " .. prenom .. " !")
end

direBonjour("Alex")   -- Salut Alex !
direBonjour("Sarah")  -- Salut Sarah !

-- Plusieurs paramètres
local function additionner(a, b)
    print(a + b)
end

additionner(10, 5)  -- 15

Retourner une valeur (return)

Parfois, on veut que la fonction nous donne un résultat au lieu de juste l'afficher. On utilise return.

return.lua 
local function carre(nombre)
    return nombre * nombre
end

local resultat = carre(5)
print(resultat)  -- 25

-- On peut l'utiliser directement
print(carre(10) + 2)  -- 102

Créez une fonction estMajeur(age) qui retourne true si l'âge est >= 18, et false sinon.

Résumé

  • Définition : local function nom() ... end
  • Appel : nom()
  • Les paramètres permettent de passer des données à la fonction
  • return permet de récupérer un résultat de la fonction
