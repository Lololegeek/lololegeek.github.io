La Boucle For

La boucle for est parfaite quand vous savez à l'avance combien de fois vous voulez répéter une action.

Syntaxe de base

Une boucle for numérique compte d'un nombre de départ jusqu'à un nombre de fin.

for_simple.lua 
-- for variable = debut, fin do
for i = 1, 5 do
    print("Numéro " .. i)
end

-- Affiche :
-- Numéro 1
-- Numéro 2
-- Numéro 3
-- Numéro 4
-- Numéro 5

La variable i

Par convention, on utilise souvent i (pour index ou itérateur), mais vous pouvez l'appeler comme vous voulez (ex: compteur).

Le Pas (Step)

Par défaut, la boucle avance de 1 en 1. Mais vous pouvez changer cela en ajoutant un troisième nombre : le pas (step).

pas.lua 
-- Compter de 2 en 2
for i = 0, 10, 2 do
    print(i)
end
-- Affiche: 0, 2, 4, 6, 8, 10

-- Compte à rebours (pas négatif)
for i = 5, 1, -1 do
    print(i .. "...")
    task.wait(1)
end
print("Décollage !")

Boucles imbriquées

On peut mettre une boucle dans une autre boucle. C'est utile pour parcourir des grilles par exemple.

imbrique.lua 
for x = 1, 2 do
    for y = 1, 2 do
        print("Coordonnées: " .. x .. ", " .. y)
    end
end

Exercice

Créez un script qui affiche la table de multiplication de 7 (de 1 à 10).

Exemple de sortie : 7 x 1 = 7, 7 x 2 = 14...

Résumé

  • for i = debut, fin do : Boucle standard (+1)
  • for i = debut, fin, pas do : Boucle avec pas personnalisé
  • Utilisez un pas négatif pour compter à l'envers
  • Contrairement à while, la boucle for ne risque pas d'être infinie (sauf si on modifie la variable i, ce qu'il ne faut pas faire !)
