La Boutique : Achat d'Items

Créez un magasin simple où les joueurs peuvent dépenser leur argent virtuel (Coins) pour acheter des objets ou des bonus.

Logique d'Achat

Le principe est simple : quand le joueur touche le bouton d'achat, on vérifie s'il a assez d'argent. Si oui, on lui retire l'argent et on lui donne l'objet.

bouton_achat.lua 
local bouton = script.Parent
local prix = 50
local nomObjet = "Epee"

-- L'objet à donner doit être dans ReplicatedStorage
local objetADonner = game.ReplicatedStorage:WaitForChild(nomObjet)
local debounce = false

bouton.Touched:Connect(function(hit)
    local player = game.Players:GetPlayerFromCharacter(hit.Parent)
    
    if player and not debounce then
        debounce = true
        
        local leaderstats = player:FindFirstChild("leaderstats")
        local money = leaderstats and leaderstats:FindFirstChild("Coins")
        
        if money and money.Value >= prix then
            -- Achat réussi
            money.Value = money.Value - prix
            
            -- Donner l'objet (copie)
            local clone = objetADonner:Clone()
            clone.Parent = player.Backpack
            
            print("Achat effectué !")
            
            -- Feedback visuel (vert)
            bouton.BrickColor = BrickColor.new("Lime green")
            wait(1)
            bouton.BrickColor = BrickColor.new("Bright red")
        else
            -- Pas assez d'argent (rouge)
            print("Pas assez d'argent !")
            bouton.BrickColor = BrickColor.new("Really red")
            wait(1)
            bouton.BrickColor = BrickColor.new("Bright red")
        end
        
        debounce = false
    end
end)

Backpack ?

Le Backpack est l'inventaire du joueur. Tout objet de type Tool mis dedans apparaîtra dans la barre d'outils en bas de l'écran.

Sécurisation (RemoteEvents)

Si vous faites une boutique via une interface graphique (GUI), vous DEVEZ utiliser un RemoteEvent. Ne faites jamais de transactions d'argent dans un LocalScript (le joueur pourrait tricher !).

1. LocalScript (Interface)

-- Dans le bouton GUI
local bouton = script.Parent
local event = game.ReplicatedStorage:WaitForChild("AchatEvent")

bouton.MouseButton1Click:Connect(function()
    -- Demander au serveur d'acheter l'objet "Epee"
    event:FireServer("Epee")
end)

2. Script Serveur (Sécurité)

-- Dans ServerScriptService
local event = Instance.new("RemoteEvent")
event.Name = "AchatEvent"
event.Parent = game.ReplicatedStorage

local prixObjets = {
    ["Epee"] = 50,
    ["Potion"] = 20
}

event.OnServerEvent:Connect(function(player, nomObjet)
    local prix = prixObjets[nomObjet]
    
    if prix then
        local coins = player.leaderstats.Coins
        
        -- Le serveur vérifie si le joueur a VRAIMENT l'argent
        if coins.Value >= prix then
            coins.Value = coins.Value - prix
            -- Donner l'objet...
        end
    end
end)

Résumé

  • Vérifier money.Value >= prix avant de vendre
  • Soustraire le prix : money.Value = money.Value - prix
  • Cloner l'objet dans player.Backpack
  • Utiliser RemoteEvents pour les achats via GUI pour éviter la triche
