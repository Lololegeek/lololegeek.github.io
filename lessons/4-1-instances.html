Les Instances

Tout dans Roblox est une Instance ! Des Parts aux Scripts en passant par les Players, apprenez à créer et manipuler ces objets fondamentaux.

Qu'est-ce qu'une Instance ?

Une Instance est un objet dans Roblox. Chaque élément que vous voyez dans l'Explorer (Parts, Scripts, Models, etc.) est une Instance. Elles ont toutes :

  • Un Parent - L'objet qui les contient
  • Un Name - Leur nom unique
  • Des Propriétés - Leurs caractéristiques (couleur, taille...)
  • Des Méthodes - Des actions qu'elles peuvent faire

Créer une Instance

Pour créer une nouvelle Instance, on utilise Instance.new() :

creation.lua 
-- Créer une nouvelle Part
local maPart = Instance.new("Part")

-- Définir ses propriétés
maPart.Name = "SuperPart"
maPart.Size = Vector3.new(4, 4, 4)
maPart.Color = Color3.fromRGB(255, 0, 0)
maPart.Anchored = true

-- IMPORTANT : Définir le Parent pour l'ajouter au jeu
maPart.Parent = workspace

N'oubliez pas le Parent !

Une Instance sans Parent n'apparaît pas dans le jeu. Définissez toujours .Parent après avoir configuré les propriétés.

Accéder aux Instances

acces.lua 
-- Accès direct par le nom
local baseplate = workspace.Baseplate

-- FindFirstChild - Retourne nil si pas trouvé (PLUS SÛR)
local part = workspace:FindFirstChild("MaPart")
if part then
    print("Trouvé !")
end

-- WaitForChild - Attend que l'objet existe
local gui = player.PlayerGui:WaitForChild("MonGui")

-- GetChildren - Récupère tous les enfants
for _, child in ipairs(workspace:GetChildren()) do
    print(child.Name)
end

Bonne pratique

Utilisez FindFirstChild() ou WaitForChild() plutôt que l'accès direct pour éviter les erreurs quand un objet n'existe pas encore.

Cloner et Détruire

clone_destroy.lua 
-- Cloner une Instance existante
local original = workspace.MaPart
local copie = original:Clone()
copie.Position = original.Position + Vector3.new(5, 0, 0)
copie.Parent = workspace

-- Détruire une Instance
copie:Destroy()  -- Supprime l'objet définitivement

-- Vérifier si une Instance existe encore
if copie.Parent == nil then
    print("L'objet a été détruit")
end

Résumé

  • Instance.new("Type") crée une nouvelle Instance
  • Toujours définir .Parent pour que l'objet apparaisse
  • :FindFirstChild() cherche sans erreur si absent
  • :WaitForChild() attend que l'objet existe
  • :Clone() duplique une Instance
  • :Destroy() supprime définitivement
