Les Éléments Universels

Découvrez les briques fondamentales pour construire toutes vos interfaces Roblox.

Roblox Studio Requis

Cette leçon utilise l'API de Roblox. Ce code ne fonctionnera que dans Roblox Studio ou notre éditeur intégré, et non sur un interpréteur Lua PC standard.

La Frame (Le Conteneur)

La Frame est l'élément de base. Elle ne fait "rien" par elle-même, mais elle sert de conteneur pour organiser les autres éléments.

Propriétés clés :

  • BackgroundColor3 : La couleur du fond.
  • BackgroundTransparency : 0 (opaque) à 1 (invisible).
  • BorderSizePixel : Épaisseur de la bordure (souvent mise à 0 pour un style moderne).

Éléments de Texte

TextLabel

Utilisé pour afficher du texte fixe (ex: titres, labels de score).

TextButton

Un bouton avec du texte qui peut être cliqué par l'utilisateur.

TextScaled : Cochez cette option pour que le texte s'adapte automatiquement à la taille de la boîte !

Éléments d'Image

Pour ajouter des icônes, des logos ou des textures de fond, on utilise :

  • ImageLabel : Simple affichage d'image.
  • ImageButton : Une image cliquable (ex: une icône "Fermer").
Exemple : Changer une image par script 
local logo = script.Parent.Logo
logo.Image = "rbxassetid://123456789" -- ID de l'image sur Roblox

Moderniser l'UI (UICorner & UIGradient)

Roblox propose des objets "modificateurs" à placer *à l'intérieur* de vos Frames ou Boutons :

UICorner

Arrondit les bords de n'importe quel élément UI.

UIGradient

Ajoute des dégradés de couleurs dynamiques.

UIStroke

Ajoute des bordures précises ou des contours au texte.
