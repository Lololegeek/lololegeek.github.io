Animations & Tweens

Maîtrisez TweenService pour créer des transitions fluides et professionnelles.

Roblox Studio Requis

Cette leçon utilise l'API de Roblox. Ce code ne fonctionnera que dans Roblox Studio ou notre éditeur intégré, et non sur un interpréteur Lua PC standard.

Qu'est-ce qu'un Tween ?

Un "Tween" est le fait de faire varier une propriété d'une Valeur A à une Valeur B de manière fluide sur un temps donné. Au lieu de changer instantanément la couleur d'un bouton, vous le faites dégradé doucement sur 0.5 seconde.

Utiliser TweenService

Voici la structure standard d'un Tween :

Animer une Frame 
local TweenService = game:GetService("TweenService")
local frame = script.Parent

-- 1. Les réglages (Temps, Style d'accélération)
local info = TweenInfo.new(
    0.5, -- Temps en secondes
    Enum.EasingStyle.Quart, -- Style (Elastic, Bounce, Sine...)
    Enum.EasingDirection.Out -- Direction
)

-- 2. Les propriétés à atteindre
local goal = {
    Position = UDim2.new(0.5, 0, 0.5, 0), -- Centre de l'écran
    BackgroundColor3 = Color3.fromRGB(255, 100, 100)
}

-- 3. Création et lecture
local tween = TweenService:Create(frame, info, goal)
tween:Play()

Easing Styles (Styles de courbe)

La fluidité dépend du style choisi :

  • Linear : Vitesse constante (un peu robotique).
  • Sine : Démarrage et arrêt plus doux.
  • Elastic/Bounce : Ajoute un effet de rebond très "cartoon".
  • Quart/Quint : Très professionnel, idéal pour les menus modernes.
